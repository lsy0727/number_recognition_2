### 함수 : bounding_img, morph
Mat bounding_img(Mat img) : img영상의 객체부분을 추출하기 위한 함수

Mat morph(Mat img) : img영상의 객체에 모폴로지 연산을 수행하는 함수

### 기능 : contour, center, stroke
contour : 외곽선의 갯수를 구한다.

center : 무게중심의 위치를 비율(%)로 구한다.

stroke : 획 수를 구한다.
